# py-pdftools

`py-pdftools` is a Python library for high-level PDF transformations backed by
Apache PDFBox and compiled to a native library with GraalVM Native Image.

The initial release is focused on adding invisible, searchable OCR text layers
to existing PDFs. The project is currently under active development; see
[`SPECS.md`](SPECS.md) for the contract and [`PROGRESS.md`](PROGRESS.md) for the
implementation status.
